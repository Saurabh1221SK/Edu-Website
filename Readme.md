# 📚 Edu Website

A modern and responsive educational website built with **HTML5**, **CSS3**, and **JavaScript (ES6)**. It offers an intuitive and engaging platform where students can explore courses, read educational blogs, learn about instructors, and connect through a dedicated contact page.

## 🌐 Live Demo

**Website:** https://saurabh1221sk.github.io/Edu-Website/

## ✨ Features

* 📱 Fully responsive design
* 🎨 Modern and clean user interface
* 🏠 Home page
* ℹ️ About page
* 📚 Courses page
* 👨‍🏫 Teachers page
* 📝 Blog page
* 📩 Contact page
* 📖 Detailed course information
* 👥 Instructor showcase
* ⚡ Fast loading and lightweight
* 🌍 Cross-browser compatibility

## 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and responsive layouts
* **JavaScript (ES6)** – Interactive functionality
